# Programming Projects

This repository contains programming projects developed using C and Java.

## Projects

### Bank Management System (C)

A console-based banking application developed as part of the Programming (1) course at Alexandria University.

#### Features
- User authentication
- Account creation and management
- Search and advanced search
- Deposit, withdrawal, and money transfer
- Account activation/deactivation
- Transaction history
- File-based data persistence
- Account sorting and reporting
- Input validation and error handling

> The original university project specification is included in this repository as a PDF.

---

### FSM Reduction Using Implication Table (Java)

A Java implementation of the FSM minimization algorithm using the implication table method.

#### Features
- DFA state minimization
- Implication table construction
- Equivalent state detection
- Generation of the minimized FSM

> This project was developed independently for learning purposes.
